# Prototype Pattern

###Definition

Specify the kinds of objects to create using a prototypical instance, and create new objects by copying this prototype.

# Solution Diagram

![image](https://drive.google.com/uc?export=view&id=1LQorsoHfk2uIZz2rix97AAdwgXdt0cB6)

• Which object to create is specified at run-time.<br/>
– Which object to create is specified at run-time by copying a prototype object (prototype.clone()).<br/>
– Prototypes can be added and removed dynamically at run-time.<br/>

### Design Problems

• Which object to create is specified at compile-time.<br/>
– Which object to create is specified at compile-time by a direct constructor call (new Product1()).
